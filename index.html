FeryyDevv — Auto Deploy

Upload ZIP hasil build → Deploy ke Vercel → Tampilkan link

ZIP should contain static build folder (build/ or dist/).

Menunggu tindakan

Petunjuk singkat

  1. Deploy project ini di Vercel sebagai satu project (index.html + /api).
  2. Set environment variable VERCEL_TOKEN di Vercel (Project → Settings → Environment Variables).
  3. Buka halaman, pilih ZIP hasil build, klik Upload & Deploy.
  4. Hasil deploy akan muncul sebagai link di bawah.

Keamanan

Jangan masukkan token di chat — simpan sebagai Environment Variable di dashboard Vercel.
